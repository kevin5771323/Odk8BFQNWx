# 项目简介

## 核心功能
- 基于YOLOv8的下水管道实例分割检测系统，支持5类目标检测：Concrete Mat、Concrete weight、Leakage、Pipe、Pipe coupling
- 提供完整源码、1000张标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供界面功能：实时显示目标位置、目标总数、置信度信息；支持自定义置信度和IOU阈值；可选择显示标签和原图；支持选择不同检测模型；批量检测功能

## 适用场景
- 下水管道检测与维护
- 基础设施安全监测
- 管道系统自动化巡检
- 工程质量评估

## 技术特点
- 开发环境：Pycharm+Anaconda或VSCode+Anaconda
- 技术栈：Python3.9、opencv-python、PyQt5、ultralytics、torch1.9
- 已训练好模型，配置环境后可直接使用
- 包含UI界面源文件和图标，提供完整的项目运行教程
